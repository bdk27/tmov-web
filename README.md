# TMOV(前端) - 影視資訊與訂票平台

TMOV 是一個基於 Nuxt 3 構建的現代化影視資訊與線上訂票平台。整合了 TMDB API 提供豐富的電影、影集與人物資料，並結合 Spring Boot 後端實現完整的會員系統、收藏、觀看紀錄與模擬訂票功能。

## 功能

#### 影視探索

- 首頁儀表板：包含動態背景圖 (Hero)、本日/本週趨勢、現正熱映及各類熱門推薦（電影、影集、動畫、綜藝）。
- 搜尋：支援跨類別（電影、電視、人物）的即時搜尋功能。
- 詳細資訊頁：
  - 完整呈現電影/影集資訊、評分、預告派、劇照。
  - 演員與劇組陣容列表。
  - 相關系列作品與推薦內容。
  - 留言討論區：支援即時留言與載入更多功能。

#### 會員中心

- 身份驗證：
  - 支援 Email/密碼 註冊與登入。
  - Google 第三方登入整合。
  - JWT Token 權限管理。
- 個人化功能：
  - 我的收藏：可收藏電影、影集或人物，支援分類篩選。
  - 觀看紀錄：自動記錄瀏覽過的內容，並提供一鍵清除功能。
- 帳號管理：
  - 編輯個人資料（暱稱、性別、生日、手機、地址）。
  - 大頭貼上傳：支援圖片預覽與上傳。
  - 修改密碼。

#### 線上訂票系統

- 訂票流程：
  - 選擇場次：依據日期與地區篩選 TMOV 影城場次，支援多種廳別（數位、IMAX、4DX、Gold Class）。
  - 視覺化選位：互動式座位圖，自動識別走道與已售座位。
  - 訂單確認：模擬結帳流程。
- 我的票夾：
  - 查看歷史訂單狀態（待付款、已付款、已取消）。
  - 倒數計時：待付款訂單顯示剩餘時間（30 分鐘內未付款自動取消）。
  - 模擬線上付款與取消訂單操作。

## 專案技術

- Nuxt 3 (Vue 3 + TypeScript)
- Nuxt UI
- Tailwind CSS
- Pinia
